# 📋 Listify - React Todo App

Listify is a simple and elegant Todo List application built with React. It allows users to add, view, and delete their daily tasks with ease. The project uses modern React features like Hooks and Browser Local Storage to provide a responsive and persistent user experience.

---

## 🚀 Features

- ✅ Add new todos with a title and description
- 🗑️ Delete individual todos
- 💾 Persistent state using localStorage
- 📱 Responsive design
- 🧭 Client-side routing with React Router
- 🌙 Clean and minimal UI
